submitForm ( ) _global_
Currently, this is a simplified version of 'saveForm' for situations where localStorage is only used as a backup, without saved data loading functionality. It only allows validated forms to be submitted. 'Submitted' in this case actually still means it is saved to localStorage first and then forced to upload. If uploading fails the user can continue working on a blank form. Uploading will be attempted at intervals until it succeeds.
